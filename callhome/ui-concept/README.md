# UI concept

A standalone design prototype of the call screen — open `index.html` (call screen, by Elle) or `incoming-call.html` (frosted incoming-call card with quick-decline, day/night, by Matt) in any browser. No backend, no APIs: this is what a Callhome call *looks and feels* like (paired avatars, breathing ambience, frosted caption bubbles, warm/cool themes).

The wired-up PWA (invite polling, VAD, streaming TTS) ships in v0.2 — see the roadmap.

*Call screen designed by Elle. Incoming-call card by Matt. One phone, two hands.*
